Introduction to HTML

HTML stands for HyperText Markup Language. It is used to create web pages.

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to HTML!</h1>
</body>
</html>

HTML Elements

HTML elements are the building blocks of HTML pages.

<p>This is a paragraph.</p>
<a href="https://example.com">This is a link</a>

HTML Attributes

Attributes provide additional information about an element.

<a href="https://example.com" target="_blank">Visit Example</a>

HTML Headings

HTML provides six levels of headings.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

HTML Paragraphs

Paragraphs are defined with the <p> tag.

<p>This is a paragraph.</p>

HTML Links

Links allow navigation between web pages.

<a href="https://example.com">Click here</a>

HTML Images

Images are embedded using the <img> tag.

<img src="image.jpg" alt="Description">

HTML Tables

Tables organize data into rows and columns.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
</table>

HTML Forms

Forms allow user input.

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

HTML Media

HTML supports audio and video elements.

<video controls>
    <source src="video.mp4" type="video/mp4">
</video>
