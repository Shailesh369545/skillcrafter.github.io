CSS Colors

CSS allows you to define colors using various formats, including named colors, HEX, RGB, and HSL.

1. Named Colors

You can use predefined color names in CSS.

                
p {
    color: red;
}

2. HEX Colors

HEX color codes define colors using six-digit hexadecimal values.

                
p {
    color: #ff5733;
}

3. RGB Colors

RGB defines colors using Red, Green, and Blue values.

                
p {
    color: rgb(255, 87, 51);
}

4. RGBA (with transparency)

RGBA allows you to set an opacity level.

                
p {
    color: rgba(255, 87, 51, 0.5);
}

5. HSL Colors

HSL stands for Hue, Saturation, and Lightness.

                
p {
    color: hsl(9, 100%, 64%);
}
