CSS Fonts

CSS allows us to control the appearance of text using font properties.

Font Properties

  • font-family - Specifies the font
  • font-size - Controls text size
  • font-weight - Defines boldness
  • font-style - Italic or normal
  • letter-spacing - Adjusts space between letters
  • word-spacing - Adjusts space between words

Example Code

                
p {
    font-family: 'Arial', sans-serif;
    font-size: 20px;
    font-weight: bold;
    font-style: italic;
    letter-spacing: 2px;
    word-spacing: 5px;
}
