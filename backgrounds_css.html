CSS Backgrounds

CSS backgrounds allow you to apply colors, images, and gradients to elements.

1. Background Color

Apply a solid background color using the background-color property.

                
body {
    background-color: lightblue;
}

2. Background Image

Set an image as the background.

                
body {
    background-image: url('background.jpg');
    background-repeat: no-repeat;
    background-size: cover;
}

3. Background Repeat

Control how the background image repeats.

                
body {
    background-image: url('pattern.png');
    background-repeat: repeat-x; /* Repeat only horizontally */
}

4. Background Position

Adjust the position of the background image.

                
body {
    background-image: url('background.jpg');
    background-position: center top;
}

5. Background Attachment

Fix the background while scrolling.

                
body {
    background-image: url('background.jpg');
    background-attachment: fixed;
}

6. Background Gradient

Create a gradient background.

                
body {
    background: linear-gradient(to right, red, yellow);
}
