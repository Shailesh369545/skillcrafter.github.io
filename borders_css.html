CSS Borders

CSS borders allow you to define the edges of an element with different styles, colors, and widths.

1. Basic Border

                
.box {
    border: 2px solid black;
}
This is a bordered box.

2. Border Types

You can change the border style using dotted, dashed, solid, double, groove, ridge, inset, or outset.

                
.dashed-border {
    border: 3px dashed red;
}
.dotted-border {
    border: 3px dotted blue;
}
Dashed Border
Dotted Border

3. Rounded Borders

You can create rounded corners using border-radius.

                
.rounded {
    border: 2px solid black;
    border-radius: 15px;
}
Rounded Borders

4. Border Color

You can customize the border color using any valid CSS color.

                
.colored-border {
    border: 3px solid green;
}
Colored Border
