What is an HTML Form?

HTML Forms are used to collect user input and send it to a server for processing.

Basic HTML Form Example

Here is a simple example of an HTML form:

<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Submit">
</form>

Common Input Types

HTML supports multiple input types for different user interactions.

<input type="text" placeholder="Text">
<input type="email" placeholder="Email">
<input type="password" placeholder="Password">
<input type="number" placeholder="Number">
<input type="checkbox"> Checkbox
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
