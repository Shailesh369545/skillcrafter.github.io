What is an HTML Table?

An HTML table is used to organize data into rows and columns.

Basic Table Structure

Here is a simple HTML table:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
        <td>USA</td>
    </tr>
    <tr>
        <td>Maria</td>
        <td>30</td>
        <td>Canada</td>
    </tr>
</table>
