What is Text Formatting?

HTML provides various tags to format text and make it bold, italic, underlined, or more.

Basic Text Formatting

Here are some commonly used HTML text formatting tags:

<b>Bold Text</b> 
<i>Italic Text</i> 
<u>Underlined Text</u> 
<strong>Strong Text (Important)</strong> 
<em>Emphasized Text</em>
