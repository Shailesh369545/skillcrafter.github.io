HTML Basics

HTML documents use elements like headings, paragraphs, and comments.

Headings

HTML provides six levels of headings:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Paragraphs

<p>This is a paragraph of text.</p>

HTML Comments

<!-- This is a comment in HTML -->
