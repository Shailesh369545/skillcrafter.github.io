Understanding CSS Syntax

CSS follows a simple structure: selectors, properties, and values.

Example:

This CSS rule sets the text color of all paragraphs to blue:

                
p {
    color: blue;
    font-size: 16px;
}

Explanation:

  • Selector (p) - Targets all

    elements.

  • Property (color) - Defines the text color.
  • Value (blue) - Sets the text color to blue.
