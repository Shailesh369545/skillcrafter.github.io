CSS Grid

CSS Grid is a powerful layout system that allows you to create complex web layouts easily.

Grid Properties

  • display: grid; - Enables grid layout
  • grid-template-columns - Defines column structure
  • grid-template-rows - Defines row structure
  • gap - Adds spacing between grid items
  • justify-items - Aligns items horizontally

Example Code

                
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
    gap: 20px;
}

.grid-item {
    background-color: #007BFF;
    color: white;
    padding: 20px;
    text-align: center;
    font-size: 1.2rem;
    border-radius: 5px;
}
