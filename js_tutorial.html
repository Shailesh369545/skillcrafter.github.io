Introduction to JavaScript

JavaScript is a versatile programming language used for web development.

// This is a simple JavaScript alert
alert("Welcome to JavaScript!");

JavaScript Variables

Variables store data values in JavaScript.

let name = "SkillCrafters"; 
console.log(name);

Data Types in JavaScript

JavaScript supports different data types like strings, numbers, and objects.

let age = 25; // Number
let isStudent = true; // Boolean

JavaScript Functions

Functions allow code to be reusable.

function greet() {
    return "Hello, World!";
}
console.log(greet());

Event Handling

JavaScript can handle user interactions using events.

document.getElementById("btn").addEventListener("click", function() {
    alert("Button clicked!");
});

DOM Manipulation

Modify HTML elements using JavaScript.

document.getElementById("demo").innerText = "Hello, JavaScript!";

Asynchronous JavaScript

Asynchronous programming allows JavaScript to execute tasks without blocking the main thread.

setTimeout(() => {
    console.log("Executed after 2 seconds");
}, 2000);

Promises & Async/Await

Promises help handle asynchronous operations.

function fetchData() {
    return new Promise(resolve => {
        setTimeout(() => resolve("Data loaded"), 2000);
    });
}

fetchData().then(console.log);

ES6 Features

ES6 introduced features like arrow functions, template literals, and let/const.

const sum = (a, b) => a + b;
console.log(sum(5, 3));
