What is HTML5?

HTML5 is the latest version of HyperText Markup Language, designed to improve functionality, support multimedia, and enhance user experience on the web.

New Semantic Elements

HTML5 introduced several new semantic elements to define different sections of a webpage:

<header> - Defines a header section
<nav> - Defines navigation links
<section> - Defines a section in the document
<article> - Defines an article or independent content
<aside> - Defines content aside from the main content
<footer> - Defines a footer section

Example Usage

<header>
    <h1>Welcome to My Website</h1>
</header>

<nav>
    <a href="#">Home</a> | <a href="#">About</a> | <a href="#">Contact</a>
</nav>

<section>
    <h2>Main Section</h2>
    <p>This is a section of content.</p>
</section>

<footer>
    <p>Copyright 2025</p>
</footer>

New Form Input Types

HTML5 introduced new input types for better form validation and user experience:

<input type="email"> - For email input
<input type="tel"> - For telephone numbers
<input type="url"> - For URLs
<input type="date"> - For date selection
<input type="range"> - For range sliders
<input type="color"> - For color picker
