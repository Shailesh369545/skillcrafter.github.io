CSS Animations

CSS animations allow elements to change from one style to another smoothly.

Key Animation Properties

  • @keyframes - Defines the animation
  • animation-name - Assigns the keyframes to an element
  • animation-duration - Defines how long the animation lasts
  • animation-timing-function - Sets the speed curve
  • animation-delay - Delays animation start
  • animation-iteration-count - Repeats the animation

Example Animation

                
@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-20px);
    }
}

.animated-box {
    width: 100px;
    height: 100px;
    background-color: #007BFF;
    color: white;
    text-align: center;
    line-height: 100px;
    border-radius: 10px;
    animation: bounce 1s infinite;
}
