CSS Text Formatting

CSS provides various properties to style text, such as font size, color, alignment, and spacing.

Text Formatting Properties

  • color - Sets the text color
  • font-size - Defines text size
  • text-align - Aligns text
  • line-height - Sets space between lines
  • text-transform - Changes text case
  • text-shadow - Adds shadow effects

Example Code

                
p {
    color: blue;
    font-size: 18px;
    text-align: center;
    line-height: 1.6;
