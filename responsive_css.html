CSS Responsive Design

Responsive Design ensures that web pages look good on all devices (desktops, tablets, and phones).

Key Concepts in Responsive Design

  • Media Queries - Apply styles based on screen size.
  • Flexible Grid - Use percentage widths instead of fixed units.
  • Fluid Images - Images should resize based on screen width.

Example: Responsive Layout

                
.container {
    display: flex;
    flex-wrap: wrap;
}

.box {
    width: 30%;
    margin: 1%;
    padding: 20px;
    background-color: #007BFF;
    color: white;
    text-align: center;
    border-radius: 10px;
}

/* Responsive Adjustments */
@media (max-width: 768px) {
    .box {
        width: 45%;
    }
}

@media (max-width: 480px) {
    .box {
        width: 100%;
    }
}
