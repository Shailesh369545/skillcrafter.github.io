CSS Margins & Padding

Margins and padding control the space around and inside an element.

1. Margin

Margins define space *outside* the element.

                
.margin-box {
    margin: 20px;
    border: 2px solid blue;
    padding: 10px;
}
This box has a margin.

2. Padding

Padding defines space *inside* the element.

                
.padding-box {
    padding: 20px;
    border: 2px solid red;
}
This box has padding.

3. Individual Margins & Padding

You can set values separately for top, right, bottom, and left.

                
.custom-margin {
    margin: 10px 20px 30px 40px;
    border: 2px solid green;
}

.custom-padding {
    padding: 5px 10px 15px 20px;
    border: 2px solid purple;
}
