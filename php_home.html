Introduction to PHP

PHP is a server-side scripting language designed for web development but also used as a general-purpose language.

<?php echo "Hello, World!"; ?>

PHP Syntax

PHP scripts are executed on the server and have a special syntax.

<?php
// This is a comment
echo "PHP Syntax Example";
?>

PHP Variables

Variables in PHP start with a `$` sign.

<?php
$name = "SkillCrafters";
echo "Welcome to " . $name;
?>

PHP Data Types

PHP supports various data types like integers, floats, strings, arrays, etc.

<?php
$intVar = 100;
$floatVar = 10.5;
$stringVar = "Hello";
?>

PHP Operators

Operators are used to perform operations on variables and values.

<?php
$a = 10;
$b = 5;
echo $a + $b; // Output: 15
?>

PHP Loops

Loops in PHP help to execute a block of code multiple times.

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Number: $i <br>";
}
?>

PHP Functions

Functions are reusable blocks of code.

<?php
function greet($name) {
    return "Hello, " . $name;
}
echo greet("SkillCrafters");
?>

Handling Forms in PHP

Forms allow user interaction with the server.

<form method="post">
    Name: <input type="text" name="name">
    <input type="submit">
</form>

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    echo "Hello, " . $_POST['name'];
}
?>

PHP Database Connection

PHP connects to databases using MySQLi or PDO.

<?php
$conn = new mysqli("localhost", "root", "", "test_db");
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully";
?>
