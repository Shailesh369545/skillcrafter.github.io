Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML elements.

p { color: blue; }

CSS Selectors

Selectors target HTML elements to apply styles.

h1 { text-align: center; }

Box Model

The Box Model includes margins, borders, padding, and content.

div { padding: 10px; border: 1px solid black; }

Flexbox

Flexbox helps with responsive layouts.

.container { display: flex; justify-content: space-between; }

CSS Grid

CSS Grid allows two-dimensional layouts.

.grid { display: grid; grid-template-columns: 1fr 1fr; }

CSS Animations

Animations make elements move smoothly.

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }

CSS Transitions

Transitions animate changes smoothly.

button { transition: background 0.3s ease; }

Responsive Design

Make websites adaptable to different screen sizes.

@media (max-width: 600px) { body { background: lightgray; } }
